Decision Tree v2: Caption detection with text ceiling

Figure caption only triggers image fallback when text is sparse enough that the figure is likely on this page

# Thresholds
TEXT_MIN = 150 # below → FLATTENED
TEXT_MAX = 600 # above + caption → cross-ref to another page, ignore

if len(text) > TEXT_MIN:
  if has_tables:
    → STRUCTURED_TEXT
  elif images:
    → TEXT_WITH_IMAGES # always, image is clearly here
  elif has_caps and len(text) < TEXT_MAX:
    → TEXT_WITH_IMAGES # sparse text + caption → likely vector graphic
  else: # dense text, any captions are cross-refs
    → PURE_TEXT
else:
    → FLATTENED
✓ Dense text + "see Figure 12"
text_length = 1,200 chars
has_caps = true (cross-reference)
images = []

len(text) > TEXT_MAX → PURE_TEXT
No false LLaVA call. Figure is on another page.
✓ Sparse text + "Figure 3: Timing diagram"
text_length = 280 chars (caption + title)
has_caps = true
images = [] ← vector graphic, not listed

len(text) < TEXT_MAX → TEXT_WITH_IMAGES
Full page render → LLaVA. Catches the vector.
TEXT_MAX = 600 is approximate — a page with 600+ chars of running text is almost certainly a prose/spec page cross-referencing a figure elsewhere. A page with 150–600 chars is likely a diagram page with a short caption and label text. Both thresholds are config-tunable.

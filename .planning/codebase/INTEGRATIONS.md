# External Integrations

**Analysis Date:** 2026-03-25

## APIs & External Services

**Qdrant Vector Database:**
- Service: Qdrant (local Docker container or remote instance)
- What it's used for: Vector storage and semantic search over embedded documents
  - SDK/Client: `qdrant-client` 1.7+
  - Configuration: `cfg["qdrant_url"]` (required field in `.carta/config.yaml`)
  - Default: `http://localhost:6333`
  - Environment variable override: `CARTA_QDRANT_URL`
  - Collections created: `{project_name}_doc`, `{project_name}_session`, `{project_name}_quirk`

**Ollama Embedding API:**
- Service: Ollama (local service for running embedding models)
- What it's used for: Computing vector embeddings for documents and search queries
  - SDK/Client: HTTP API via `requests` library
  - Configuration: `cfg["embed"]["ollama_url"]` and `cfg["embed"]["ollama_model"]`
  - Default URL: `http://localhost:11434`
  - Default model: `nomic-embed-text:latest` (768-dim vectors)
  - Alternative model for judging: `phi3.5-mini` (used if `proactive_recall.ollama_judge: true`)
  - Environment variable override: `CARTA_OLLAMA_URL`
  - API endpoint: `POST {ollama_url}/api/embeddings` in `carta/embed/embed.py:27`
  - Auth: None (local service)

## Data Storage

**Databases:**
- **Qdrant** (Vector Database)
  - Connection: HTTP REST API at `cfg["qdrant_url"]`
  - Client: `qdrant-client` Python SDK
  - Vector configuration: 768-dimensional COSINE distance
  - Collections:
    - `{project_name}_doc` - Embedded documents and reference material
    - `{project_name}_session` - Session memory (cross-request context)
    - `{project_name}_quirk` - Project-specific quirks and patterns
  - Idempotent upserts via deterministic UUID generation: `hashlib.md5(f"{slug}:{chunk_index}")`

**File Storage:**
- Local filesystem only
- Supported formats:
  - `.pdf` - Via PyMUPDF extraction in `carta/embed/parse.py:extract_pdf_text()`
  - `.md` / `.yaml` - Inline docs with frontmatter
  - `.wav` / audio transcripts - Via `docs/audio/` path in config
- Reference material location: `docs/reference/` (configurable)
- Metadata sidecars: `*.embed-meta.yaml` files co-located with source files

**Caching:**
- No explicit caching service (local process cache only)
- Ollama models cached locally on disk by Ollama runtime

## Process Lock Mechanism

**Embed Process Lock:**
- File: `.carta/embed.lock`
- Purpose: Ensure only one embedding process runs simultaneously
- Implementation: Atomic file creation + PID tracking in `carta/cli.py:_acquire_embed_lock()`
- Stale lock detection: Checks if PID is still alive via `os.kill(pid, 0)`

    In `carta/embed/parse.py`, add two functions after the existing `extract_pdf_text`:

    `_strip_frontmatter(text: str) -> tuple[str, dict]`:
    - Use `re.match(r'^---\n(.*?\n)---\n', text, re.DOTALL)` to detect YAML frontmatter

    `extract_markdown_text(md_path: Path, max_tokens: int = 800) -> tuple[list[dict], dict]`:
    - Read file with `md_path.read_text(encoding="utf-8")`
    - Call `_strip_frontmatter()` to get clean text and frontmatter_meta dict
    - Split on heading boundaries: `re.split(r'(?=^#{2,3}\s)', text, flags=re.MULTILINE)`
    - For each section: extract heading from first line if it starts with `#`, remainder is text body
    - Return shape MUST match extract_pdf_text: `[{"page": i+1, "text": section_text, "headings": [heading]}]`
    - Also return frontmatter_meta as second element of tuple
    - Empty sections (whitespace only) should be skipped
    - If a section has no heading (e.g., text before first ##), use heading `"(intro)"`

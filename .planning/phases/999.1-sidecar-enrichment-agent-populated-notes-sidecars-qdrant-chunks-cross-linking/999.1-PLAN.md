<feature>
  <name>carta/embed/lifecycle.py — versioning and chunk lifecycle primitives</name>
  <files>carta/embed/lifecycle.py, carta/tests/test_lifecycle.py</files>
  <behavior>
    HASH-01: compute_file_hash(path: Path) -> str
      - Markdown (.md): read_bytes(), replace(b'\r\n', b'\n'), sha256.hexdigest()
      - PDF (.pdf): read_bytes() raw, sha256.hexdigest() (no normalization)
      - Other text: treat as markdown (LF normalization)
      - Returns lowercase hex string

    HASH-02: needs_rehash(file_path: Path, sidecar: dict) -> bool
      - Returns True if sidecar has no file_mtime key
      - Returns False if os.path.getmtime(file_path) == sidecar["file_mtime"]
      - Returns True if mtime values differ (float comparison)

    LIFECYCLE-01: mark_sidecar_stale(client, collection_name: str, sidecar_id: str, now: datetime) -> None
      - Calls client.set_payload(collection_name, payload={stale_as_of: now.isoformat(), status: "stale"}, points=Filter(...))
      - Filter: FieldCondition(key="sidecar_id", match=MatchValue(value=sidecar_id))
      - Does NOT call set_payload if sidecar_id is empty string

    LIFECYCLE-02: cleanup_expired_orphans(client, collection_name: str, ttl_days: int, now: datetime) -> int
      - Builds cutoff = now - timedelta(days=ttl_days)
      - Scrolls with DatetimeRange(lt=cutoff) on orphaned_at field
      - Deletes matched point IDs via client.delete(collection_name, points_selector=ids)
      - Returns count of deleted points
      - Returns 0 if scroll returns empty list (no error)
      - Protected collections: caller responsibility (lifecycle.py does not filter by doc_type — callers skip protected collections)

    LIFECYCLE-03: is_protected_doc_type(doc_type: str) -> bool
      - Returns True for: "quirk", "bug-note", "helpful-note"
      - Returns False for all others ("doc", "datasheet", "manual", "session", unknown strings)

    STALE-01: check_stale_alert(stale_count: int, total_count: int, threshold: float) -> str | None
      - Returns None if total_count == 0
      - Returns None if stale_count / total_count < threshold
      - Returns a warning string if stale_count / total_count >= threshold
      - Warning string includes the percentage and "carta embed --force-stale" hint
      - threshold is a fraction (0.30 = 30%)
  </behavior>

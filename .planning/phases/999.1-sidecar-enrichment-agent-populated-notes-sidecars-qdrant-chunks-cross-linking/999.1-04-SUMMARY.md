### Task 2: carta_embed scope parameter (MCP-01, MCP-02, MCP-03)

**Status:** COMPLETE

Implemented extended `carta_embed` MCP tool with three scope paths:

**MCP-01 (scope='file')**
- Signature: `carta_embed(scope="file", path="docs/x.pdf")`
- Behavior: Resolves path relative to repo_root if needed, calls `run_embed_file`
- Returns: `{"status": "ok", "scope": "file", "chunks": N}`
- Error: Returns `{"error": "invalid_request", "detail": "path is required..."}` if path=None

**MCP-02 (scope='stale')**
- Signature: `carta_embed(scope="stale")`
- Behavior: Calls `discover_stale_files(repo_root)` → list of Path, re-embeds each via `run_embed_file`
- Returns: `{"status": "ok", "scope": "stale", "reembedded": N}`
- Handles errors per file without crashing on first failure

**MCP-03 (scope='all')**
- Signature: `carta_embed(scope="all")` [default]
- Behavior: Preserves existing behavior via `run_embed(repo_root, cfg)`
- Returns: Full collection embed result (existing format)

**Backward Compatibility Guard**
- If `scope not in ("stale", "file", "all")` and `path is None`:
  - Treat scope value as path
  - Set scope="file"
  - Allows old-style callers: `carta_embed("docs/x.pdf")` → redirects to file scope

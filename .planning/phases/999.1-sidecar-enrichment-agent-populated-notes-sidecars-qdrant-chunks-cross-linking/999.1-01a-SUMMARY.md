## Implementation Details

### Task 1: `compute_file_hash` (HASH-01)

**Function Signature:**
```python
def compute_file_hash(path: Path) -> str
```

**Behavior:**
- Markdown (`.md`) files: `read_bytes()` → replace `\r\n` with `\n` → `sha256.hexdigest()`
- PDF (`.pdf`) files: `read_bytes()` raw, no normalization → `sha256.hexdigest()`
- Other text files: treated as markdown (LF-normalized)
- Returns lowercase hexadecimal string (64 characters)

**Key Insight:** Line-ending normalization is critical for markdown (which humans read cross-platform) but must be skipped for binary PDF to preserve exact content hashing.

### Task 2: `needs_rehash` (HASH-02)

**Function Signature:**
```python
def needs_rehash(file_path: Path, sidecar: dict) -> bool
```

**Behavior:**
- Returns `True` if sidecar has no `"file_mtime"` key (file never hashed before)
- Returns `False` if `os.path.getmtime(file_path) == sidecar["file_mtime"]` (no change)
- Returns `True` if mtime values differ (file modified)

**Key Insight:** The mtime fast-path is the primary performance optimization — if mtime hasn't changed, the file content hasn't changed, so we skip the expensive SHA256 computation entirely.

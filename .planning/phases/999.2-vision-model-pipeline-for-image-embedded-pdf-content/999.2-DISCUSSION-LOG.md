## Image Format / Resolution

| Option | Description | Selected |
|--------|-------------|----------|
| Scale to max 1024px, PNG | Good balance, preserves detail | |
| Full resolution, PNG | Maximum fidelity | |
| Scale to max 512px, JPEG | Fastest, smallest payload | |
| Native res, 2048px ceiling, PNG | Extract at native, only downscale outliers | ✓ |

**User's choice:** Native resolution with 2048px ceiling, always PNG
**Notes:** User raised concern about "damaging" images via blanket downscaling. After discussion: extract embedded images at native resolution via `extract_image(xref)`, cap at 2048px max dimension (preserving aspect ratio) only for outliers. Convert to PNG (lossless). For pages with no embedded images but vector drawing content, render page at 150 DPI as fallback. Both strategies active.

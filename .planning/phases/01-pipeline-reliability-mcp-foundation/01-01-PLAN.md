1. Call `get_embedding()` inside try/except — on failure, print warning and `continue` (skip that chunk, do NOT add to batch)
2. On success, append PointStruct to `batch`

3. Extract the per-file processing block (lines 122-151, inside the `try:` in the for loop) into a new function:
```python
def _embed_one_file(file_path: Path, file_info: dict, cfg: dict, client, repo_root: Path, max_tokens: int, overlap_fraction: float, verbose: bool) -> tuple[int, dict]:
```

   This function does: extract PDF text, chunk, build enriched chunks, call `upsert_chunks`, return `(count, sidecar_updates_dict)`. It uses `if verbose: print(...)` for progress.
4. In the main loop of `run_embed()`, wrap `_embed_one_file` with:
```python
with concurrent.futures.ThreadPoolExecutor(max_workers=1) as executor:
    future = executor.submit(_embed_one_file, file_path, file_info, cfg, client, repo_root, max_tokens, overlap_fraction, verbose)
```

**PIPE-05 — Sidecar heal pass (per D-05):**

In `carta/embed/pipeline.py`, add function `_heal_sidecar_current_paths(repo_root: Path, verbose: bool = False) -> int:`:
```python
def _heal_sidecar_current_paths(repo_root: Path, verbose: bool = False) -> int:
    healed = 0
    for sidecar_path in repo_root.rglob("*.embed-meta.yaml"):
        data = read_sidecar(sidecar_path)
        if data is None or "current_path" in data:
            continue
        stem = sidecar_path.name.replace(".embed-meta.yaml", "")
        for ext in _SUPPORTED_EXTENSIONS:
            candidate = sidecar_path.parent / f"{stem}{ext}"
            if candidate.exists():
                data["current_path"] = str(candidate.relative_to(repo_root))
                _update_sidecar(sidecar_path, data)
                healed += 1
                break
    if verbose and healed:
        print(f"carta embed: healed {healed} sidecar(s) missing current_path", flush=True)
    return healed
```

Call `_heal_sidecar_current_paths(repo_root, verbose=verbose)` in `run_embed()` after `ensure_collection` and before `discover_pending_files`.
